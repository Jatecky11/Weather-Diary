# Weather Diary 🌦️ 

Автор: [Гусманов Эмиль]

### Описание
Консольное приложение для ведения дневника погоды. Позволяет фиксировать температуру, типы погоды и визуализировать изменения через графики.

### Функционал
- Сохранение данных в JSON.
- Валидация ввода дат и чисел.
- Построение линейного графика температуры (Matplotlib).
- Инкапсуляция данных и использование наследования.
