## 📝  О проекте
**Weather Diary** — это консольное приложение на языке Python, предназначенное для личного мониторинга погодных условий. Проект создан для тех, кто хочет вести структурированный учет метеорологических данных без использования сложных веб-интерфейсов.

### Зачем нужен этот проект?
1.  **Аналитика:** Отслеживание динамики изменения температуры в вашем регионе.
2.  **Архивация:** Возможность хранить историю наблюдений в легком формате JSON.
3.  **Визуализация:** Быстрая генерация графиков для наглядного представления температурных скачков.
4.  **Образование:** Демонстрация принципов объектно-ориентированного программирования (ООП) и работы с внешними библиотеками анализа данных.

---

## 🚀  Основные возможности
* **Гибкий учет:** Добавление записей о погоде с разделением на "Солнечно" и "Осадки" (использование наследования классов).
* **Безопасность данных:** Инкапсуляция полей модели данных и автоматическое сохранение в JSON-файл.
* **Умная валидация:** Проверка вводимых дат, температурных диапазонов и типов данных для предотвращения сбоев программы.
* **Графический анализ:** Визуализация истории температур с помощью библиотеки `matplotlib`.

---

## 🛠️  Технологический стек
* **Язык:** Python 3.10+
* **Библиотеки:** * `matplotlib` (визуализация графиков)
    * `datetime` (работа с временными метками)
    * `json` (хранение данных)
